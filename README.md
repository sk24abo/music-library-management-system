# Music Library Management System (Java)

## Overview
Object-oriented Java application developed as part of a Large-Scale Programming module (79%). The system models a music library with playlists and song collections, focusing on modularity, maintainability, and structured program design.

## Features
- Song management using encapsulated class design
- Playlist creation and modification
- Playlist collection management
- Input validation and error handling
- Menu-driven user interaction

## Technical Concepts Applied
- Object-Oriented Programming (encapsulation, modular design)
- Multi-class architecture
- Separation of concerns
- Clean code structuring for maintainability

## How to Run
Compile and run the main class (e.g., `MusicParty.java`) using any Java IDE (NetBeans, IntelliJ) or via terminal:

javac *.java
java MusicParty
